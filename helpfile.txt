> ***Welcome*** 

Welcome to the text-based RPG life simulator that is DiscordLife

> ***Commands***

 - `!create` -> Start a new game!

 - `!profile` -> Shows all the details of your character 

 - `+age` -> Progresses the game by incrementing the age of the character [Note the + prefix]

 - `!surrender` -> Ends the current character : (

 - `!help` -> The trusty command to bring up this message at anytime!
